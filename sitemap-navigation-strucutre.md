
## Full Sitemap & Navigation Structure

The site is organized into logical sections to support a guided journey and reference material. Below is the full sitemap with all primary routes:

- **Home** (/) - Landing page welcoming the user and directing them into the guide.
- **Roadmap Hub** (/roadmap) - Overview of all stages in the onboarding roadmap.
- **Stage Page** (/roadmap/\[stageSlug\]) - e.g. /roadmap/day-0-1. Details for a specific stage and list of steps.
  - **Step Page** (/roadmap/\[stageSlug\]/\[stepSlug\]) - e.g. /roadmap/week-1/prayer. Core instructional content for a single step.
- **Topics Hub** (/topics) - A list of additional topic guides for deeper dives or common questions.
- **Topic Page** (/topics/\[topicSlug\]) - e.g. /topics/ramadan. Standalone guide on a specific subject.
- **Glossary** (/glossary) - A-Z list of Islamic terms and definitions.
- **Resources Hub** (/resources) - Collection of curated external resources.
- **Masjid Finder** (/resources/find-masjid) - Tool to find nearby mosques with search/filter.
- **Ramadan Guide** (/ramadan) - Separate page or alias for Ramadan content (critical, direct access).
- **Mental Health Guide** (/mental-health) - Separate page or alias for Mental Health topic.
- **Sources** (/sources) - Bibliography or citations used throughout content.
- **About** (/about) - Information about Revert Guide project, mission, team.
- **Accessibility Statement** (/accessibility) - WCAG compliance details.
- **Privacy Policy** (/privacy) - Legal page outlining user privacy.
- **Terms of Use** (/terms) - Terms and conditions.
- **404 Page** (/404) - Implicit; handles unknown routes with friendly message.

**Desktop Top Navigation:**
- **Home** (logo clickable to home)
- **Roadmap** (main link; consider dropdown listing each stage for quick jumping)
- **Topics** (main link; dropdown if many topics)
- **Glossary** (separate top-level link for importance)
- **Resources** (main item; dropdown: Find Masjid, External Links)
- **About** (main item; dropdown: About, Accessibility, Privacy, Terms, Sources)

**Navigation Rationale:**
- Desktop users see more options. Roadmap highlighted (core content).
- Topics and Glossary next (address user needs).
- Resources secondary (mostly external).
- About/More last (site info).
- Logo left (convention) always clickable to home.
- Nav bar sticky (fixed) at top on desktop. Bottom nav fixed on mobile for easy one-handed use.

**Breadcrumbs:**
- On all content pages showing hierarchy (e.g., "Roadmap / Week 1 / Step 3").
- Each part clickable for quick navigation.
- Helps users orient and navigate internally.

**Footer:**
- Site footer on all pages with links to legal pages (Accessibility, Privacy, Terms, Sources) if not in top nav.
- Note like "© 2026 Revert Guide".
- On mobile: minimal footer (bottom nav used); placed after content.

**Navigation Design Goal:**
- Balance guidance (Roadmap encourages sequential use) with exploration (Topics/Glossary allow immediate answers).
- Small number of primary options (not overwhelming).
- Intuitive icons/labels recognizable to non-tech-savvy users.
- Achieves goals: "I know exactly where to start" and "I won't be overwhelmed."
